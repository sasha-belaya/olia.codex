## Концепция 1: "Нейро-Брутализм"

**Design Movement**: Цифровой брутализм с элементами глитч-арта и киберпанк-эстетики

**Core Principles**:
1. **Асимметрия и хаос**: Намеренно нарушенная сетка, диагональные срезы секций, наложение элементов
2. **Контрастная типографика**: Крупные заголовки Montserrat Black (120-180px) с мелким телом Manrope Regular
3. **Технологичная грубость**: Резкие переходы, видимые границы, монохромные блоки с акцентом розового
4. **Цифровая аутентичность**: Показываем "швы" дизайна — видимые grid lines, ASCII-элементы, code snippets

**Color Philosophy**: 
Доминирующий чёрный (#000000) как символ "чистого листа" перед обучением. Розовый (#ff007f) — резкие акценты для CTA и ключевых элементов, создающие визуальное напряжение. Белый текст на чёрном — максимальная читаемость без компромиссов.

**Layout Paradigm**: 
Разбитая сетка с диагональными срезами между секциями (clip-path). Модули курса — вертикальная timeline с асимметричными карточками. Тарифы — три колонки разной высоты с розовыми рамками разной толщины.

**Signature Elements**:
1. **Глитч-эффект на заголовках**: Анимированное раздвоение текста с RGB-смещением
2. **Диагональные срезы**: clip-path: polygon() для создания угловых переходов между секциями
3. **ASCII-декор**: Использование символов █▓▒░ для декоративных элементов

**Interaction Philosophy**:
Резкие, мгновенные переходы. Hover на кнопках — инверсия цвета без transition. Scroll — появление элементов блоками, без плавных fade-in.

**Animation**:
- Заголовки: glitch-эффект при входе в viewport (transform + opacity + text-shadow)
- CTA-кнопки: пульсация розовой тени (box-shadow animation)
- Модули: появление слева/справа поочерёдно (translateX)
- Таймер: мигающие двоеточия

**Typography System**:
- H1: Montserrat Black 140px / line-height 1.1 / letter-spacing -0.02em
- H2: Montserrat Bold 72px / line-height 1.2
- H3: Montserrat SemiBold 48px
- Body: Manrope Regular 18px / line-height 1.6
- Accent: Inter 28pt для цифр и таймера

## Концепция 2: "Минимализм Будущего"

**Design Movement**: Швейцарский модернизм встречается с японским минимализмом и AI-эстетикой

**Core Principles**:
1. **Дышащее пространство**: Огромные отступы (120-200px между секциями), воздух как главный элемент
2. **Типографическая иерархия**: Один шрифт (Montserrat) в 5 начертаниях, размер = важность
3. **Монохромная база + хирургический акцент**: 95% чёрно-белое, 5% розовый только для действий
4. **Сеточная точность**: Строгая 12-колоночная сетка, всё выравнено по baseline grid

**Color Philosophy**:
Белый (#ffffff) фон как символ ясности и понимания. Чёрный (#000000) текст — абсолютная читаемость. Розовый (#ff007f) — исключительно для CTA и активных состояний, создаёт фокус внимания. Светло-серый (#f2f2f2) для разделителей.

**Layout Paradigm**:
Центрированная вертикальная ось с максимальной шириной 800px для текста. Тарифы — горизонтальный scroll на мобиле, три колонки на десктопе с равной высотой. Модули — нумерованный список с огромными цифрами (200px) на фоне.

**Signature Elements**:
1. **Огромные цифры-watermark**: Номера модулей как фоновые элементы (opacity 0.03)
2. **Тонкие линии-разделители**: 1px solid #f2f2f2 для визуального ритма
3. **Розовый dot**: Маленькая точка (8px) рядом с активными элементами

**Interaction Philosophy**:
Плавность и предсказуемость. Все transition — 300ms ease-out. Hover — лёгкое изменение opacity (0.7). Клик — мгновенная обратная связь через scale(0.98).

**Animation**:
- Появление контента: fade-in + translateY(20px) с задержкой между элементами 100ms
- CTA: hover — розовая тень растёт (box-shadow spread)
- Scroll: progress bar (1px розовая линия) в топе страницы
- Таймер: плавная смена цифр с flip-эффектом

**Typography System**:
- H1: Montserrat Light 96px / line-height 1.1 / letter-spacing -0.03em
- H2: Montserrat Regular 64px / line-height 1.2
- H3: Montserrat Medium 40px
- Body: Montserrat Regular 20px / line-height 1.8
- Accent: Montserrat Bold для выделений, Inter 28pt для таймера

## Концепция 3: "Нео-Мемфис"

**Design Movement**: Постмодернизм 80-х через призму современного веб-дизайна и нейро-эстетики

**Core Principles**:
1. **Игривая геометрия**: Круги, треугольники, волнистые линии как декоративные элементы
2. **Многослойность**: Overlapping элементов, тени, глубина через z-index
3. **Смелая типографика**: Наклонные заголовки, вращение текста, mixed case
4. **Энергичная палитра**: Все цвета палитры в игре, розовый — главная звезда

**Color Philosophy**:
Розовый (#ff007f) как энергия и креативность — используется щедро (фоны секций, кнопки, акценты). Чёрный (#000000) и тёмно-серый (#222222) для контраста и заземления. Белый (#ffffff) для читаемости. Светло-серый (#f2f2f2) для мягких фонов. Цвет = эмоция, не просто оформление.

**Layout Paradigm**:
Асимметричная сетка с "плавающими" элементами. Hero — диагональный split (розовый фон слева, белый справа). Модули — карточки с rotation (-2deg, 1deg, -1deg) для динамики. Тарифы — карточки с цветными тенями (розовый, чёрный).

**Signature Elements**:
1. **Волнистые разделители**: SVG-волны между секциями (розовые, чёрные)
2. **Геометрические декорации**: Полупрозрачные круги и треугольники на фоне
3. **Цветные тени**: box-shadow с розовым оттенком для глубины

**Interaction Philosophy**:
Игривость и отзывчивость. Hover — лёгкое вращение (rotate 2deg) и подъём (translateY -4px). Клик — "bounce" эффект. Scroll — параллакс для декоративных элементов.

**Animation**:
- Hero: элементы влетают с разных сторон (translateX/Y) с bounce easing
- Модули: карточки появляются с rotation и scale
- CTA: пульсация + hover увеличение (scale 1.05)
- Декор: медленное вращение геометрических фигур (360deg за 20s)
- Таймер: цифры меняются с flip + цвет мигает розовый/чёрный

**Typography System**:
- H1: Montserrat ExtraBold 120px / line-height 1.0 / transform: rotate(-2deg)
- H2: Montserrat Bold 68px / line-height 1.1 / mixed case
- H3: Manrope Bold 44px / line-height 1.3
- Body: Manrope Regular 19px / line-height 1.7
- Accent: Inter 28pt Bold для цифр, розовый цвет

## Рекомендация

Для продающего сайта курса по нейро-контенту рекомендую **Концепцию 1: "Нейро-Брутализм"**. 

Причины:
- Отражает технологичность и современность темы (AI, нейросети)
- Выделяется среди типичных образовательных сайтов
- Создаёт эмоциональное напряжение, стимулирующее к действию
- Розовый акцент на чёрном — максимальная видимость CTA
- Асимметрия и динамика удерживают внимание

Альтернатива: Концепция 2 для более консервативной аудитории, ищущей "серьёзное" обучение.
