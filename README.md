# Оля Кодекс — Сайт AI-CREATOR

Репозиторий содержит исходный код сайта **oliacodex.ru**, включающий:

- **Основной сайт** (AI-CREATOR) — лендинг курса по нейро-контенту
- **Проект /veb** — лендинг вебинара AI CODEX

---

## Структура проекта

```
olia.codex/
├── client/              # Исходный код основного сайта (React + Vite)
├── server/              # Заглушка для серверной части (не используется)
├── shared/              # Общие константы и типы
├── veb/                 # Проект вебинара /veb
│   ├── client/          # Исходный код вебинара (React + Vite)
│   ├── server/          # Заглушка для серверной части
│   └── shared/          # Общие константы и типы
├── package.json         # Зависимости основного проекта
├── vite.config.ts       # Конфигурация Vite для основного проекта
└── README.md            # Этот файл
```

---

## Требования

- **Node.js** версии 18 или выше
- **pnpm** версии 8 или выше

Установка pnpm:

```bash
npm install -g pnpm
```

---

## Установка зависимостей

### Основной проект

```bash
pnpm install
```

### Проект /veb

```bash
cd veb
pnpm install
cd ..
```

---

## Локальная разработка

### Запуск основного сайта

```bash
pnpm dev
```

Сайт будет доступен по адресу `http://localhost:3000`

### Запуск проекта /veb

```bash
cd veb
pnpm dev
```

Проект будет доступен по адресу `http://localhost:3000`

---

## Сборка для production

### Сборка основного сайта

```bash
pnpm build
```

Готовые файлы будут в папке `dist/`

### Сборка проекта /veb

```bash
cd veb
pnpm build
cd ..
```

Готовые файлы будут в папке `veb/dist/public/`

## Развертывание на VDS

Подробная инструкция по развертыванию на VDS сервере находится в файле **DEPLOYMENT_GUIDE.md**.

Краткая версия:

1. Соберите оба проекта
2. Объедините файлы в одну структуру
3. Загрузите на сервер в `/var/www/oliacodex.ru/`
4. Настройте Nginx
5. Установите SSL-сертификат

---

## Обновление сайта на сервере

После внесения изменений и коммита в GitHub:

1. Подключитесь к серверу:
   ```bash
   ssh user@oliacodex.ru
   ```

2. Перейдите в директорию репозитория:
   ```bash
   cd /var/www/olia.codex
   ```

3. Получите последние изменения:
   ```bash
   git pull origin main
   ```

4. Соберите проекты:
   ```bash
   pnpm install
   pnpm build
   cd veb && pnpm install && pnpm build && cd ..
   ```

5. Скопируйте собранные файлы:
   ```bash
   ./deploy.sh
   ```
